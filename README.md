## ✨ Features

| Feature | Description |
|---------|-------------|
| ✅ **Create Tasks** | Add tasks with title, description, priority & due date |
| ✅ **Complete Tasks** | One-click completion with toast notifications |
| ✅ **Edit Tasks** | Update task details anytime |
| ✅ **Delete Tasks** | Remove unwanted tasks |
| ✅ **Search Tasks** | Smart search with Ctrl+K shortcut |
| ✅ **Filter Tasks** | Filter by status, priority, or category |
| 📱 **Responsive** | Works perfectly on all devices |
| 🌙 **Dark Theme** | Beautiful dark mode design |
| 📊 **Statistics** | Real-time progress tracking |
| 🏷️ **Categories** | Academic, Personal, Work, General |
| ⚡ **Priority Levels** | High, Medium, Low priority flags |
| 📅 **Due Dates** | Track deadlines with overdue alerts |
| 🎯 **Keyboard Shortcuts** | Ctrl+K to search, Escape to clear |
| 🔔 **Toast Notifications** | Instant feedback for all actions |
| 🎨 **Smooth Animations** | Beautiful UI transitions |
| ✨ **Particles Background** | Floating animated dots |
| 📊 **Stats Counter** | Numbers count up on scroll |

## 🏃‍♂️ Run Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- [Git](https://git-scm.com/)

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/leenahayat/StudentHub.git
cd StudentHub

# 2. Install dependencies
npm install

# 3. Start the server
npm start

# 4. Open in browser
# Visit: http://localhost:3000
